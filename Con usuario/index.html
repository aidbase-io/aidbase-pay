🛡️ Form Validation Rules

Smart rules applied automatically to all public forms. Toggle on/off and edit error messages. Saved to all forms instantly.

💡 Tip: When you create a new form in Form Builder, you can choose to inherit these rules automatically — the fields will get the right validations without any extra work.

✅ Field Validations

Set rules for each field in all public forms — Sign, Register, Emergency Kits.
Changes are saved and applied automatically to all forms.

ℹ️ How does this work? Click to learn
Each field can have its own rules:
⭐ Required — The person must fill this field. If empty, they can't submit the form.
🔢 Numbers only — Only digits are allowed (0–9). Good for ID number, phone, etc.
🔤 Letters only — Only letters allowed. Good for names.
📏 Min digits — The minimum number of characters required. Example: ID must be at least 9 digits.
📏 Max digits — The maximum number of characters allowed.
💬 Error message — What the person sees if they fill it wrong. If empty, a default message is shown.

Example: ID field → Required ✓, Numbers only ✓, Min 9, Max 9 → "ID must be exactly 9 digits"
